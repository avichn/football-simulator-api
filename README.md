A modular, CRUD-based football simulation API designed to manage teams, players, matches, and tournaments while supporting probabilistic match simulations driven by team performance metrics. The system models team strength using weighted attributes such as attack, midfield, defense, recent form, and externally synchronized ranking data. It is structured to serve as the backend foundation for a full-scale football simulator, enabling dynamic tournament generation, match simulation, standings calculation, and integration with third-party sports data providers.
